# Statistical Methods & Data Visualization — Coursework Collection

## Overview
This repository contains selected homework assignments and assessments 
from Data Modeling & Visualization, taught by Dr. Clair at Saint Louis 
University. Work is completed in R using R Markdown and R scripts. Files 
are included to demonstrate applied statistical reasoning and fluency with 
R across a range of methods and datasets.

## Methods Demonstrated
- Exploratory data analysis and descriptive statistics
- Hypothesis testing (t-tests, Welch two-sample)
- Bootstrap resampling and simulation
- Simple and multiple linear regression
- Log and logit transformations, residual diagnostics
- Logistic regression and GLMs (binomial family)
- Odds ratios and probability prediction
- Data visualization with ggplot2 and GGally

## Datasets Used
All datasets are publicly available through R packages or course-provided 
sources. Sources include:

- `storms` (dplyr) — NOAA Atlantic storm tracking data
- `crickets` (modeldata) — cricket chirp rate by species and temperature
- `COMPAS` recidivism dataset — criminal risk score and recidivism outcomes
- Sleuth3 package datasets — flea beetles, metabolic rates, mammal brain 
weight, bee pollen, Drosophila wing size, deforestation
